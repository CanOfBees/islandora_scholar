# Islandora Bibliography

## Introduction

Allows users to maintain/export their own bibliography of citations.

## Requirements

This module requires the following modules/libraries:

* [Islandora](https://github.com/islandora/islandora)
* [Tuque](https://github.com/islandora/tuque)
* [islandora_bookmark](https://github.com/Islandora/islandora_bookmark)
* csl
* citeproc

## Installation

Install as usual, see [this](https://drupal.org/documentation/install/modules-themes/modules-7) for further information.

## Configuration

Set the paths for `example` and `module` in Administration » Islandora » MODULE (admin/islandora/module).

Include a screenshot of configuration page.
